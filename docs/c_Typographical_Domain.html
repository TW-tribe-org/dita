Typographic (highlighting) domain

The elements in the typographic domain are used to describe styling characteristics. They are primarily intended as the basis for specialisation. Avoid using these elements.
Semantic elements should always be used in preference to typographic elements.

Rationale

Never use these elements when a semantically specific element is available. These elements are not intended for use by specializers, and are intended solely for use by authors when no semantically appropriate element is available and a formatting effect is required.

Typographic domain elements

The typographic domain elements each have a specific styling purpose.

Element Name Semantic Purpose
b bold (strong)
i italic (emphasis)
u underlined
tt teletype (typewriter)
sup superscript
sub subscript
