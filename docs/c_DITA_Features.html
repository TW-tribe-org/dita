DITA features

DITA incorporates features designed to improve the quality and efficiency of documentation.

DITA's features include:
  • modularity
  • structured authoring (reduces authoring time, increases analysis time)
  • information typing
  • minimalism
  • inheritance
  • specialisation
  • single-source
  • topic-based
  • metadata
  • conditional processing
  • component publishing
  • task-orientation
  • content re-use
  • translation-friendly structure
